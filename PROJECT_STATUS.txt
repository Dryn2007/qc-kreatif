🚀 QUICK START

Local Development:
 $ php artisan serve

 → Press F12 → Ctrl+Shift+M for mobile view

Deploy to Production:
 1. Set environment variables
 2. Run migrations: php artisan migrate
 3. Deploy to Vercel or your host

Export Routes:
 GET /export/excel?minggu=Minggu%201  → Download CSV
 GET /export/pdf?minggu=Minggu%201    → View/print PDF
